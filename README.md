A botanical-themed interactive scrapbook for pinning and reliving your favorite moments across the Little Red Dot.

# Features
1. Interactive Memory Mapping: Pin your special moments to specific coordinates across Singapore using a custom-styled Leaflet map.
2. Polaroid Reveal: Click on pins to reveal your memories in a nostalgic Polaroid-style overlay, complete with handwritten captions and washi-tape details.
3. "Start Our Story" (Time Travel Mode): A cinematic experience that takes you on a chronological journey through your pinned memories, automatically panning the map and revealing each moment in sequence.
4. Botanical Aesthetic: A carefully crafted UI inspired by "Cottagecore" romance, featuring linen backgrounds (#F5F0EB), dried rose accents (#C4878E), and floating heart animations.
5. Local Persistence: Your memories are saved directly to your browser's localStorage, ensuring your story stays private and accessible on your device without the need for a backend database.

# Tech Stack
1) Frontend: React 19, Vite, Wouter (Routing).
2) Mapping: Leaflet & React-Leaflet.
3) Styling: Tailwind CSS v4 & Lucide React icons.
4) Animations: Framer Motion for smooth, organic transitions and floating UI elements.
5) Utility: Zod (Validation), Sonner (Toasts), and Nanoid.

# Getting Started
Clone the repository:

Bash
git clone https://github.com/spacerac00n/valentines-love-map.git
Install dependencies:

Bash
pnpm install
Run in development mode:

Bash
pnpm dev
Build for production:

Bash
pnpm build
